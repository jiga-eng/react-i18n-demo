# React-Application-With-i18next

## Getting Started

1. Clone this repo:

```sh
git clone https://github.com/jiga-eng/react-i18n-demo react-i18n-demo
cd react-i18n-demo
```

2. Install project dependencies:


```sh
yarn

# If you are using NPM
npm install
```
3. Start the project
```sh
npm start

# If you are using Yarn
yarn start
```
